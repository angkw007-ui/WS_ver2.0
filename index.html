Google Sheets 연동 설정
일정을 Google Sheets에 백업·동기화합니다.
Apps Script 설치
1
Google Sheets에서 새 스프레드시트를 만드세요.
2
확장 프로그램 → Apps Script → 아래 코드 붙여넣기 → 배포 → 웹 앱 → 액세스: 모든 사용자
function doGet(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName("events") || ss.insertSheet("events"); return ContentService .createTextOutput(JSON.stringify(sh.getDataRange().getValues())) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName("events") || ss.insertSheet("events"); sh.clearContents(); JSON.parse(e.postData.contents).forEach(r => sh.appendRow(r)); return ContentService.createTextOutput("OK"); }
3
배포 후 생성된 웹 앱 URL을 아래에 입력하세요.
